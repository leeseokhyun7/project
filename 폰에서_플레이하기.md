# 📱 네온 서바이버즈 — 폰에서 플레이하기

이 폴더는 이제 **오프라인 설치형 웹앱(PWA)** 으로 준비돼 있습니다.
아래 파일들을 **한 폴더째** 인터넷에 올리면, 폰에서 앱처럼 설치해 출퇴근 시간에 즐길 수 있어요.

포함 파일: `index.html`, `manifest.webmanifest`, `sw.js`, `icon.svg`, `icon-192.png`, `icon-512.png`

---

## 방법 A. 넷리파이 드롭 — 가장 쉬움 (계정 없이 1분)

1. 폰 또는 PC 브라우저에서 **https://app.netlify.com/drop** 접속
2. 위 **파일 7개를 한꺼번에** 그 페이지로 끌어다 놓기
   - PC라면: 이 폴더를 열고 파일 전체를 선택해 드래그
   - (폴더 통째로 드래그해도 됩니다)
3. 잠시 후 `https://랜덤이름.netlify.app` 주소가 생깁니다 → 이게 게임 주소
4. 이 주소를 폰에서 열면 바로 플레이 ▶

> 주소를 그대로 두면 계속 쓸 수 있습니다. 무료 계정으로 로그인하면 주소를 예쁘게 바꾸거나 관리할 수 있어요.

---

## 방법 B. 깃허브 페이지스 (주소를 오래 고정하고 싶을 때)

1. https://github.com 가입 → 새 저장소(Repository) 생성 (Public)
2. 위 파일 7개 업로드 (Add file → Upload files)
3. Settings → Pages → Branch를 `main` / 루트로 설정 후 저장
4. 몇 분 뒤 `https://아이디.github.io/저장소이름/` 주소가 발급됩니다

---

## 📲 폰에 "앱"으로 설치하기 (홈 화면 추가)

게임 주소를 폰 브라우저에서 연 다음:

- **아이폰 (Safari)**: 공유 버튼 → **"홈 화면에 추가"**
- **안드로이드 (Chrome)**: 우측 위 ⋮ 메뉴 → **"앱 설치"** 또는 **"홈 화면에 추가"**

그러면 홈 화면에 아이콘이 생기고, **주소창 없는 전체화면**으로 실행됩니다.

### 오프라인(신호 없는 지하철 구간)에서도 되나요?
네. 설치 후 **와이파이/LTE가 되는 곳에서 한 번 실행**해 두면, `sw.js`가 게임을 폰에 저장해 두어 이후 **신호가 없어도** 실행됩니다.

## 조작법
- **이동**: 화면 아무 곳이나 손가락으로 끌면 나타나는 가상 조이스틱
- **공격**: 자동 발사 — 이동하며 살아남고, 레벨업 때 강화를 고르세요
- 세로/가로 모두 지원됩니다

## 팁
- 무언가 이상하면 브라우저 **새로고침**으로 최신 버전을 받으세요.
- 게임을 수정하면 파일을 다시 올린 뒤, 폰에서 앱을 껐다 켜면 갱신됩니다.
